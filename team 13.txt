USN:24BCAR0525
Name:Vansitha Jain
Algorithm:
Step1:Start
Step2:Assign values to a,b and c
Step3:Compare If a is less than or equal to b and c, then smallest = a
Step4:Else if b is less than or equal to a and c, then smallest = b
Step5:Else set smallest = c
Step6:Print smallest
Step7:Stop

Code:
a = int(input("Enter the first value:"))
b = int(input("Enter the second value:"))
c = int(input("Enter the third value:"))
if a<=b and a<=c:
  smallest = a
elif b<=a and b<=c:
  smallest = b
else:
  smallest = c
print(smallest, "is the smallest")


Output:
Enter the first value:22
Enter the second value:44
Enter the third value:10
10 is the smallest
